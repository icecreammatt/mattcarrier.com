Virtual High School

Posted on Jan 7, 2007
The virtual high school or Windsor High school map was a project I did in my senior year for school. (2005) My goal was to create the whole school in a 3d environment which could be explored in the Half-Life 2 game engine. I figured the project would take between 80 and 100 hours. It ended up taking over 200 hours. This was a great learning experience for me. While it did end up taking me so long to do it, I enjoyed almost every minute of it.

Virtual High School Install Guide

Posted on Jan 7, 2007
First you need Steam. http://www.steampowered.com Then you need to either buy a copy of Half-Life 2 or The Orange Box to run the mod. This can be picked up retail for between 20 and 50 dollars. It can also be purchased directly from steam. To install copy the “Virtual High School” folder (copy the folder that is in the same folder as the readme.txt file. Don’t copy the dated folder.) to your SourceMods folder in your steam directory.
